# TAKlib
A Python3 library to emulate a TAK Server for use with wintak, atak, civtak, itak or any other TAK devices

## Requirements
- Python 3

## Usage Example
```python
from TAKlib import TAK
TAKSock = TAK("172.21.1.166", 8087)
```
